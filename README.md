# Portfolio: Software Development & Educational Technology

Selamat datang di repositori portofolio saya. Sebagai seorang praktisi pendidikan dengan latar belakang **Informatika**, saya berfokus pada pengembangan solusi perangkat lunak yang menjembatani kompleksitas teknis dengan kebutuhan instruksional. Proyek-proyek di bawah ini mendemonstrasikan keahlian saya dalam **Python**, **Web Development (Flask)**, **Data Science/AI**, dan **Desktop GUI Development**.

## 🚀 Proyek 1: Edu-Performance Analytics Portal

**Dashboard Analitik Performa Siswa Berbasis Web**

* **Teknologi Utama**: Python, Flask, SQL (SQLite/PostgreSQL), Pandas, Plotly.
* **Deskripsi**: Platform dashboard yang dirancang untuk membantu pengajar memvisualiskan tren nilai secara otomatis dan melakukan deteksi dini terhadap siswa yang berisiko tertinggal secara akademik.
* **Fitur Utama**:
* **Automated Data Ingestion**: Fitur unggah file CSV untuk pemrosesan data nilai secara instan.
* **Interactive Visualization**: Grafik tren performa menggunakan Plotly yang terintegrasi dalam antarmuka Flask.
* **Historical Tracking**: Sistem database terstruktur untuk menyimpan rekam jejak perkembangan per kelas dari waktu ke waktu.


* **Konteks Strategis**: Proyek ini mentransformasi proses pengelolaan data manual yang memakan waktu menjadi sistem otomatis yang memberikan wawasan berbasis data secara real-time bagi pendidik.

## 🧠 Proyek 2: LexiGrade - AI Text Complexity Classifier

**Klasifikasi Tingkat Kesulitan Teks Berbasis Machine Learning**

* **Teknologi Utama**: Python, Scikit-learn, NLP (Natural Language Processing), Jupyter Notebook, Flask.
* **Deskripsi**: Alat berbasis kecerdasan buatan (AI) yang mengklasifikasikan teks bacaan ke dalam kategori level pendidikan (SD, SMP, atau SMA) berdasarkan kompleksitas linguistiknya.
* **Fitur Utama**:
* **NLP Pipeline**: Implementasi *pre-processing* teks termasuk *tokenizing* dan *stopword removal*.
* **Supervised Learning**: Model klasifikasi menggunakan algoritma *Random Forest* atau *SVM* yang dilatih dengan dataset kurikulum resmi.
* **Predictive API**: *Endpoint* API sederhana yang menerima input teks dan memberikan prediksi level secara instan.


* **Konteks Strategis**: Memanfaatkan keahlian di bidang NLP untuk menjawab tantangan guru dalam menyusun bahan ajar dan soal evaluasi yang tepat sasaran sesuai usia mental siswa.

## 💻 Proyek 3: Algorithmic Logic Visualizer

**Aplikasi Desktop Visualisasi Logika Algoritma**

* **Teknologi Utama**: Python, PyQt5 / Tkinter.
* **Deskripsi**: Aplikasi desktop interaktif yang memvisualisasikan cara kerja algoritma fundamental (seperti *Bubble Sort* atau *Search*) secara *step-by-step* untuk membantu pemahaman konsep abstrak.
* **Fitur Utama**:
* **Animation Control**: Kontrol GUI untuk mengatur kecepatan animasi dan variasi input data secara dinamis.
* **Memory Visualization**: Visualisasi "live" pergerakan variabel dan perubahan struktur data di dalam memori.
* **Educational Export**: Opsi untuk mengekspor alur logika (pseudocode) ke dalam format file teks.


* **Konteks Strategis**: Proyek ini menonjolkan kemampuan dalam menyederhanakan konsep pemrograman yang kompleks menjadi representasi visual yang intuitif dan mudah dipelajari.
